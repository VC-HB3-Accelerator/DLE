# Шаблоны для юристов (DLE)

## Назначение

Файлы в этой папке — **не готовые договоры** и **не привязаны к конкретной стране**.
 Это **опорные материалы для юристов**: перечень обязательных пунктов и
 рекомендуемая структура, на основе которых юрист готовит **локальные договоры**:

- **автор ↔ контрибьютор** — партнёрство, роялти, IP, территория;
- **контрибьютор ↔ клиент** — B2B, регулятор (см. [client-license-contract.md](client-license-contract.md)).

**Юрист самостоятельно определяет** (для каждой стороны и юрисдикции):

- применимое право и подсудность;
- налоги, валюту, форму оплаты;
- реквизиты, подписи, госзакупки / тендеры (если применимо);
- персональные данные и отраслевое регулирование страны;
- формулировки под тип клиента (юрлицо, госорган).

**Неизменно сохраняются** (если локальный закон не улучшает положение клиента):
 см. [CONTRIBUTOR_LICENSE.md](../CONTRIBUTOR_LICENSE.md), [LICENSE.ru](../../LICENSE.ru),
 [service-terms.md](../../docs.ru/service-terms.md).

## Иерархия договоров

```
Автор ←—— договор автор ↔ контрибьютор (contributor-author-contract)
              │
              ▼
        Контрибьютор ←—— договор с клиентом (client-license-contract)
              │              ├── Приложение: LICENSE (EULA)
              │              └── Приложение: service-terms
              ▼
        Sepolia-токен = принятие EULA у правообладателя / правопреемника
```

## Аудитория клиентов

По модели DLE типичные клиенты — **B2B** и **регулируемые организации**.
 Юрист добавляет отраслевые приложения (ИБ, аудит, DPA, SLA, хранение данных)
 по требованиям заказчика и закона **своей** юрисдикции.
