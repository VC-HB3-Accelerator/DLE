## Important files for adding headers

### Frontend
- `frontend/src/main.js`
- `frontend/src/App.vue`
- All Vue components
- All CSS/SCSS styles

### Backend
- `backend/src/index.js`
- All modules and controllers
- Configuration files

### Documentation
- `README.md`
- `docs/` - all documentation files
